## Wikipedia.js

Buscador enciclopédico simple construido con HTML, CSS y JavaScript que consume la API REST de Wikipedia en español.

![screenshot](images/screenshot.png)
<hr>

**Características**

→ Búsqueda de artículos en Wikipedia. <br>
→ Manejo de páginas con desambiguación. <br>
→ Interfaz inspirada en Wikipedia. <br>
→ Menú lateral interactivo. <br>
→ Diseño responsive. <br>
→ Sin dependencias externas de JavaScript.

**Tecnologías**

→ HTML5. <br>
→ CSS3 (Custom Properties / Design Tokens). <br>
→ JavaScript (ES6+). <br>
→ Wikipedia REST API.

**API utilizada**

→ **Endpoint:** `https://es.wikipedia.org/api/rest_v1/page/summary/{término}`<br>
→ **Fuente de contenido:** Wikipedia (es)

**Accesibilidad**

→ Uso de roles semánticos.
<br>
→ Etiquetas ARIA en elementos interactivos.
<br>
→ Navegación básica por teclado.
